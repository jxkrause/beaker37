# moviesearch

Beaker37 is a Python library for recommending for movies.

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install beaker37.

```bash
pip install git+https://github.com/jxkrause/beaker37
```

## Usage

```python
import beaker37
...
```
